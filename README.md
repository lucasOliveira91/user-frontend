# Cadastro-Senha

## Descrição
Este projeto foi gerado com [yo angular generator](https://github.com/yeoman/generator-angular) versão 0.16.0. Ele é uma aplicação web para cadastro e gerenciamento de senhas.

## Stack Tecnológica
- **Front-end**: AngularJS
- **Automatização**: Grunt
- **Testes**: Karma

## Como Iniciar

### Pré-requisitos
Certifique-se de ter o Node.js e o Grunt CLI instalados globalmente em sua máquina.

### Instalação
1. Clone o repositório para sua máquina local usando `git clone <URL_DO_REPOSITÓRIO>`.
2. Navegue até a pasta do projeto usando `cd cadastro-senha`.
3. Execute `npm install` para instalar todas as dependências.
4. Execute `bower install` para instalar as dependências front-end.

### Desenvolvimento & Visualização
- Execute `grunt serve` para iniciar o servidor de desenvolvimento e visualizar o projeto em seu navegador.

### Construção
- Execute `grunt` para construir o projeto. Os arquivos construídos estarão na pasta `dist`.

## Testes
- Execute `grunt test` para executar os testes unitários com Karma.
